FroggAR

C# Unity AR Development

Frogger, but make it AR.

Features

  • Built a mobile AR game in Unity using Vuforia featuring plane detection, mid-air object placement, and a full user-driven course creation workflow.
  • Implemented frog movement logic using phone orientation, proximity checks, press-and-hold input, and a live-updating trajectory visualization.
  • Developed platform collision, landing/respawn behavior, life tracking, and final win/lose conditions; deployed the completed app to Android.
