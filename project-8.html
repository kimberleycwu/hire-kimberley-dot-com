Araman in Numbers

JavaScript HTML/CSS Web Development

An interactive data-visualization web app that pulls live data from Google Sheets via REST API and renders real-time analytics with Chart.js.

Features

  • Hand-gathered data on dice roll outcomes and modifiers across 20+ hours of gameplay content.
  • Implemented a responsive, filterable dashboard and efficient chart-update logic to improve performance and reduce memory use.
  • Developed a modular data-processing pipeline that converts raw sheet data into structured JavaScript objects and computes aggregate metrics using dynamic column mapping.
